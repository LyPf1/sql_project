# Projeto SQL – Banco de Dados de Carros

Banco de dados relacional desenvolvido em SQL (MySQL) para fins acadêmicos,
simulando a gestão de marcas de carros, modelos em inventário e clientes.
O projeto aplica conceitos como chaves primárias, chaves estrangeiras
e integridade referencial, servindo como amostra de conhecimentos básicos
em modelagem de bancos de dados.
